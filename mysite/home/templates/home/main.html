What is this project?

This site is a pet project made with help of "Django For Everybody" University of Michigan's Course lead by Charles R. Severance. It covers most of Django base features and follows architectural pattern of Model-View-Controller (Model-Template-View in Django philosophy).

Most of the work done here implements Backend Technologies, such as:
  • Linux Ubuntu OS server environment (bash)
  • MySQL server database (migrated from SQLite3)
  • Python
  • Django Framework
  • Django ORM
  • Django Template Language
  • Sessions mechanism, cookies
  • Client-side caching
  • Django Middlewares (authentification system, CSRF-token cookies)
  • BLOB Objects uploading by users, verifications
  • Social Networks log-in
Over the Front-End, some other technologies are involved:
  • HTML, CSS
  • Ruby features implementations for Python, such as Haml and SaSS preprocessors (django-hamlpy extension)
  • Bootstrap
  • JavaScript, JQuery, parallax.js

You may visit any of Django Apps I have created on this site using top navigation bar. Enjoy!
